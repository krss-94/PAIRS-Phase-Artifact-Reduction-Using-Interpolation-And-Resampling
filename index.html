PAIRS.

Phase Artifact Reduction Using Interpolation And Re-Sampling

A multirate preprocessing wrapper that reduces intermodulation distortion in phase vocoder pitch shifting — without touching the vocoder's internals.

↓ Download demo.m Read report

Algorithm

Four-stage pipeline

The PV is untouched. PAIRS wraps it.

01 — UPSAMPLE
Polyphase FIR ×2
Creates empty sacrificial band [fs/2, 2·fs/2] above original Nyquist.
β=8, P=64, rate: 2·fs
02 — PHASE VOCODER
PV @ elevated rate
Pitch shifts at 2·fs. IMD migrates into sacrificial band. PV is unmodified.
N=2048, H=512, Hann
03 — ANTI-ALIAS LPF
64th-order FIR
Mandatory pre-decimation filter scrubs sacrificial-band IMD selectively.
ωc = π/L = π/2
04 — DOWNSAMPLE
Decimate ×2
Restores original fs. Sacrificial-band content permanently discarded.
output rate: fs

Why must the PV run at 2·fs? The Noble Identity (which allows LTI filters to commute with downsamplers) does not apply — the PV is nonlinear. Moving it after downsampling eliminates the mechanism entirely.

Quick start

Drop-in MATLAB usage

demo_quick.m 
% Load mono audio
[x, fs] = audioread('your_audio.wav');
if size(x, 2) > 1, x = mean(x, 2); end

% Pitch shift +4 semitones — PAIRS vs Raw PV
y_pairs = pairs_pitch_shift(x, fs, 4);
y_raw   = phase_vocoder(x, fs, 4);

% Evaluate
tir  = compute_tir(x, y_raw, y_pairs, fs);
dcoh = compute_phase_coherence(y_pairs, y_raw);

fprintf('TIR: %.3f  Dcoh: %+.4f\n', tir, dcoh);

% Save outputs
audiowrite('out_pairs.wav', y_pairs, fs);
audiowrite('out_raw.wav',   y_raw,   fs);

Evaluation metrics

Three complementary measures

TIR
Transient Integrity Ratio
Measures transient preservation at ±25 ms around detected signal peak. TIR > 1.0 means PAIRS wins.
TIR = RMS_err(Std PV) / RMS_err(PAIRS)
Δcoh
Inter-frame Phase Coherence Gain
Δcoh > 0 indicates more stable inter-frame phase progression in the PAIRS output vs Std PV.
Δcoh = mean_coh(PAIRS) − mean_coh(Std PV)
coh = |E[exp(j·Δφ)]| across bins
LSD
Log Spectral Distance
Frame-based spectral fidelity to an ideal reference. Lower is better. Used as safety check — zero regressions at ±3–4 st.
LSD = (1/K) · Σ_k [10·log10(S_ref[k] / S_est[k])]²
NFFT=1024, 50% overlap, Hann window
